Work

Here are some of my projects:

Basic SQL Project

This project demonstrates advanced SQL skills through comprehensive database design and complex query development. Built a complete Sales database from scratch with proper table relationships, foreign key constraints, and data population from AdventureWorks2022. Includes 10 advanced queries showcasing joins, subqueries, aggregations, window functions, and data manipulation techniques.

You can check out the source code on my GitHub repository.

Advanced SQL Project

This project demonstrates my ability to write advanced SQL queries using techniques such as window functions, CTEs (Common Table Expressions), joins, and subqueries. The script includes real-world examples that show my ability to analyze and manipulate data efficiently.

You can check out the source code on my GitHub repository.

Data Analysis with Python

In this project, I performed a full data analysis workflow using Python. I used libraries such as Pandas, Seaborn, Plotly, and Matplotlib to clean, explore, and visualize the dataset. The notebook includes visual insights, statistical summaries, and effective use of Python for real-world data storytelling.

You can check out the source code on my GitHub repository.

FIFA World Cup Data Analysis with Power BI

This project analyzes FIFA World Cup data (1930-2022) using Power BI to create four interactive dashboards covering revenue analysis, goal scoring trends, player performance, and predictive forecasting. Features advanced techniques including drill-down capabilities, custom visualizations, DAX calculations, and time-series forecasting.

You can check out the report on my Power BI dashboard.
